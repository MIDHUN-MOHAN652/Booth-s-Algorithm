# Booth's Algorithm Implementation

## Overview
Booth's Algorithm is an efficient algorithm for multiplying binary numbers, especially in the context of signed multiplication. It is based on the concept of using a single multiplication process for both positive and negative numbers, thereby simplifying the multiplication process.

## Project Description
This project implements Booth's Algorithm in a chosen programming language. The implementation focuses on its ability to handle both signed and unsigned integers, making it suitable for various applications and improving performance in digital systems.

## Features
- Efficient binary multiplication of signed integers
- Handles cases for both positive and negative numbers
- Clear and organized code structure for easier understanding
- Unit tests to validate implementation

## Implementation Details
The main aspects of the implementation include:
- Fetching input operands (both multiplicand and multiplier)
- Applying Booth's Algorithm logic to perform multiplication
- Providing output in binary and decimal formats
